Módulo 2 - Git y GitHub

Git es una herramienta para el control de versiones. Se encarga de llevar un registro de cada cambio hecho al código, por lo que el trabajo en equipo (remoto y presencial) se desenvuelve de una manera natural. Este sistema es distribuido, gratuito y de código abierto, el cual fue diseñado para manejar desde proyectos pequeños hasta ser usado en proyectos de gran envergadura sin sacrificar velocidad ni eficiencia, y a su vez siendo fácil de aprender.
